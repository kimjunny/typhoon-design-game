## Overview

ENSO Typhoon Design Game is an educational web game built for meteorology learning (KMA contest context).

Players explore typhoon environments, inspect wind-flow conditions, and solve cause–effect design problems with interactive feedback.

## Features

- Real **GFS wind-flow** visualization
- Cause–effect interactive problem flow
- Card-based typhoon environment scenarios
- Browser-ready demo on Vercel
